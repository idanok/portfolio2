## Getting Started

### Installation
1. Clone the repository:
git clone https://github.com/idanok/portfolio2.git

2. Navigate into the project folder:
cd portfolio2

3. Install dependencies:
npm install

4. Start the development server:
npm run dev

5. Open your browser at:
http://localhost:5174
